# ShopEasy E-Commerce Website

A simple e-commerce website built with HTML, CSS, and JavaScript. Features include:

- Home, Products, Cart, and Checkout pages
- Product catalog with add-to-cart functionality
- Cart management (add/remove items)
- Basic checkout form
- Responsive, modern UI

## How to Run

1. Open `index.html` in your browser.
2. Browse products, add to cart, and proceed to checkout.

## Folder Structure

- `index.html`, `products.html`, `cart.html`, `checkout.html` — Main pages
- `styles/` — CSS files
- `scripts/` — JavaScript files
- `assets/` — Images (add your own product images)

## Note
- Product images are referenced as placeholders (e.g., `assets/headphones.png`). Add your own images to the `assets` folder for best results.
